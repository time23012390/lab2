# 1. Реализация функций для вывода элементов массива
## 1.1 Постановка задачи

Необходимо реализовать функцию printArray(array), которая принимает массив и выводит элементы в консоль в формате:

```js
Element 0: value x
Element 1: value y
Element 2: value z
```

Также требуется реализовать функцию printArray1(array), которая выводит элементы массива в более кратком формате:

```js
0: x
1: y
2: z
```

Реализация функций 

```js
function printArray(arr) {
    for(let i = 0; i < arr.length; i++){
        console.log('Element ' + i + ": value " + arr[i]);
    }
}
function printArray1(arr) {
    for(let i = 0; i < arr.length; i++){
        console.log(i + ": " + arr[i]);
    }
}
let arr = [5, 8, 9]
printArray(arr);
printArray1(arr);
```

## 1.2 Реализация универсальной функции forEach

Чтобы избежать дублирования кода, реализуется универсальная функция forEach(array, callback).
Она принимает массив и функцию обратного вызова (callback) и вызывает её для каждого элемента массива.

Колбэк получает три параметра:

- элемент массива

- индекс элемента

- сам массив

Реализация функции:

```js
function forEach(array, callback){
    for(let i = 0; i < array.length; i++){
        callback(array[i], i, array);
    }
}
let shortFormat = (value, index) => {
    console.log(index + ':' + value)
}
let longFormat = (value, index) => {
    console.log('Element: ' + value + ', Index: ' + index); 
}
let arr = [5, 8, 9]
forEach(arr, shortFormat);
forEach(arr, longFormat);
```

# 2. Реализация функции map(array, callback)
## 2.1 Постановка задачи

Необходимо реализовать функцию map(array, callback), которая создает новый массив, содержащий результаты вызова функции обратного вызова (callback) для каждого элемента исходного массива.

Функция должна:

- принимать массив array

- принимать функцию обратного вызова callback

- вызывать callback(element, index, array) для каждого элемента

- возвращать новый массив той же длины

## 2.2 Реализация функции

```js
function myMap(arr, callback){
    let results = [];
    for(let i = 0; i < arr.length; i++){
        results.push(callback(arr[i], i, arr));
        
    }
    return results;
}

let multiplication = (element, index, array) => element **= 2;

let array = [4, 5, 7];

console.log(myMap(array, multiplication))
```

# 3. Реализация функции filter(array, callback)
## 3.1 Постановка задачи

Необходимо реализовать функцию filter(array, callback), которая формирует новый массив из элементов исходного массива, удовлетворяющих условию, заданному в функции обратного вызова (callback).

Функция должна:

- принимать массив array

- принимать функцию обратного вызова callback

- вызывать callback(element, index, array) для каждого элемента массива

- добавлять элемент в новый массив, если callback возвращает true

Длина результирующего массива может быть меньше или равна длине исходного.

## 3.2 Реализация функции

```js
function myFilter(arr, callback){
    let results = [];
    for(let i = 0; i < arr.length; i++){
        if(callback(arr[i], i, arr)){
            results.push(arr[i]);
        }
        else continue;
        
    }
    return results;
}

let filtration = (element, index, array) => element % 2 == 0;

let array = [4, 5, 7];

console.log(myFilter(array, filtration))
```

# 4. Реализация функции find(array, callback)
## 4.1 Постановка задачи

Необходимо реализовать функцию find(array, callback), которая возвращает первый элемент массива, удовлетворяющий условию, заданному в функции обратного вызова (callback).

Функция должна:

- принимать массив array

- принимать функцию обратного вызова callback

- вызывать callback(element, index, array) для каждого элемента массива

- вернуть первый элемент, для которого callback возвращает true

Если ни один элемент не удовлетворяет условию, функция должна вернуть undefined.

## 4.2 Реализация функции
```js
function myFind(arr, callback){
    for(let i = 0; i < arr.length; i++){
        if(callback(arr[i], i, arr)){
            return arr[i];
        }
        else continue;
        
    }
    return undefined;
}

let find = (element, index, array) => element % 2 == 0;

let array = [4, 5, 7, 8];

console.log(myFind(array, find))
```

# 5. Реализация функции some(array, callback)
## 5.1 Постановка задачи

Необходимо реализовать функцию some(array, callback), которая проверяет, существует ли хотя бы один элемент массива, удовлетворяющий условию, заданному в функции обратного вызова (callback).

Функция должна:

- принимать массив array

- принимать функцию обратного вызова callback

- вызывать callback(element, index, array) для каждого элемента массива

- возвращать true, если хотя бы для одного элемента callback вернул true

- возвращать false, если ни один элемент не удовлетворяет условию

Обход массива должен происходить слева направо и завершаться при первом найденном совпадении.

## 5.2 Реализация функции

```js
function mySome(arr, callback){
    for(let i = 0; i < arr.length; i++){
        if(callback(arr[i], i, arr)){
            return true;
        }
        else continue;
        
    }
    return false;
}
```

# 6. Реализация функции every(array, callback)
## 6.1 Постановка задачи

Необходимо реализовать функцию every(array, callback), которая проверяет, удовлетворяют ли все элементы массива условию, заданному в функции обратного вызова (callback).

Функция должна:

- принимать массив array

- принимать функцию обратного вызова callback

- вызывать callback(element, index, array) для каждого элемента массива

- возвращать true, если callback возвращает true для всех элементов массива

- возвращать false, если хотя бы один элемент не удовлетворяет условию

Обход массива должен происходить слева направо и прекращаться при первом несоответствии.

## 6.2 Реализация функции

```js
function myEvery(arr, callback){
    for(let i = 0; i < arr.length; i++){
        if(callback(arr[i], i, arr) == false){
            return false;
        }
        else continue;
        
    }
    return true;
}
```

# 7. Реализация функции reduce(array, callback, initialValue)
## 7.1 Постановка задачи

Необходимо реализовать функцию reduce(array, callback, initialValue), которая последовательно обрабатывает элементы массива, накапливая результат в аккумуляторе.

Функция должна:

- принимать массив array

- принимать функцию обратного вызова callback

- принимать необязательный параметр initialValue — начальное значение аккумулятора

Функция обратного вызова должна иметь следующий вид:

callback(accumulator, element, index, array)

Алгоритм работы функции:

- если initialValue передано, аккумулятор инициализируется этим значением, а обход начинается с первого элемента массива

- если initialValue не передано, аккумулятором становится первый элемент массива, а обход начинается со второго элемента

- если массив пустой и initialValue не передано, функция должна вернуть undefined

После обработки всех элементов функция должна вернуть итоговое значение аккумулятора.

## 7.2 Реализация функции

```js
function myReduce(arr, callback, initialValue){
    for(let i = 0; i < arr.length; i++){
        if (initialValue == undefined){
        initialValue = arr[i];
        continue;
    }
    if(initialValue == undefined && arr.length == 0)
        return undefined;
    else initialValue = callback(initialValue, arr[i], i, arr);
    }
    return initialValue;
}

let summa = (accumulator, element, index, array) => accumulator += element;

let array = [1, 2, 3, 4, 5];

console.log(myReduce(array, summa));
```

## В чем преимущества использования колбэков при работе с массивами?

Колбэки позволяют передавать поведение функции как параметр, благодаря чему можно избежать дублирования кода, так как одна функция имеет множество вариантов применения. За счет этого код становится более гибким и универсальным. Они также улучшают читаемость и структуру кода за счет чего программу легче расширять.

## Какие проблемы могут возникать при использовании колбэков и как их избежать?
При использовании колбэков могут возникать проблемы с читаемостью кода (особенно при сильной вложенности — так называемый callback hell), ошибки из-за неправильной передачи аргументов, а также потеря контекста. Чтобы этого избежать, следует писать небольшие и понятные функции, использовать стрелочные функции (они сохраняют контекст), соблюдать правильную сигнатуру колбэка и по возможности разбивать сложную логику на отдельные функции.

## Как реализовать функции map, filter, find, some, every и reduce без встроенных методов?
Функции map, filter, find, some, every и reduce можно реализовать без встроенных методов, используя обычный цикл for для перебора массива и вызов переданного callback для каждого элемента: в map создаётся новый массив с результатами работы callback, в filter добавляются только те элементы, для которых callback возвращает true, в find возвращается первый подходящий элемент, в some — true, если найден хотя бы один подходящий элемент, в every — false, если хотя бы один элемент не удовлетворяет условию, а в reduce используется аккумулятор, который на каждом шаге обновляется результатом callback, накапливая итоговое значение.
